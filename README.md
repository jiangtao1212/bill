#时光记账
功能列表：
- 主要功能：记录账单；
- 可供用户登录；
- 可注册新用户；
- 数据的保存；
- 可查看历史记录；
- 生成统计图；
- 分类，分为周，月，年查看；
- 每日支出的预算；
- 密码修改；
- 统计总数的功能；
- 可切换主题；
- 软件说明；
